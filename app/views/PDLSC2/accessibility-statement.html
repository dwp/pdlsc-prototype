Accessibility Statement

This website is run by the Department of Work and Pensions (DWP). We want as many people as possible to be able to use this website. For example that means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • navigate most of the website using just a keyboard

We've also made the website text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

The accessibility of this website

We know some parts of this website are not fully accessible:

  • not all text and icons have sufficient colour contrast between the text or icon and background colour
  • not all input fields have hint text

We carry out a range of accessibility tests, including AXE-core testing. This software automatically tests web applications and websites to highlight potential accessibility issues. The issues we found are:

  • some pages have poor colour contrast
  • some of the code is incorrectly labelled, located or duplicated
  • some page elements are incorrectly labelled in the code. These include:
    • error messages
    • buttons
    • tabs
    • input fields
    • headers

Technical information about this website's accessibility

DWP is committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications)(No.2)Accessibility Regulations 2018.

This website is partially compliant with the Web Content Accessibility Guidlines version 2.1 AA standard, due to the non-compliance listed below.

Non-compliance with the accessibility regulations

This website has been tested using manual WCAG testing and automated testing using AXE. It has not been tested using assistive technology tools. We plan to fully test the website for compliance, using assistive technology tools by September 2021.

How we tested this website

This website was last tested on 3 August July 2020. The accessibility testing was carried out by the pre-retirement digital team at DWP.

We tested our website in full and it's available at https://pdlsc.agent-service.prod.rbcdwp.net/pdlsc

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No.2) Accessibility Regulations 2018 (the 'accessibility regulations').

If you're not happy with how we respond to your complaint, contact the:

  • Equality Advisory and Support Service (EASS) if you live in England, Scotland or Wales
  • Equality Commission for Northern Ireland (ECNI) if you live in Northern Ireland
